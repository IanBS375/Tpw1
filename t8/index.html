Colores Multinavegador

Hay más de 160 colores con nombre que son compatibles con todos los navegadores modernos. Estos nombres se pueden usar directamente en el archivo de estilos, para aplicar colores a elementos sin especificar valores o modelos de colores
Por ejemplo

  1. Negro
  2. Blanco
  3. Salmón
  4. Azul
  5. Verde

Colores de un borde

Para que un elemnto HTML podamos asignarle un color a su borde son necesarias 2 propiedades: border-style y border-color, adicionalmente se puede modificar su tamaño utilizando border-width

Degradado de colores

En CSS podemos aplicar dos tipos de degradado de colores;

Degradado lineal

El degradado lineal es una transicion de un color a otro de forma lineal

Degradado radial

El degradado radial es una transicion de un color a otro de forma radial
