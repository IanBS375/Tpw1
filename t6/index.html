Lenguaje de marcado de hipertexto

Bienvenidos hoy estaremos practicando algunos de los conceptos y elementos más importantes que se han visto hasta el momento en el curso de Programación Web 1 sobre HTML además la importancia de contenedores y etiquetas semánticas en la estructuración de un sitio web

Conceptos principales de HTML

HTML

Lenguaje de marcado de hipertexto que se utiliza para la estructuración de páginas web

Elemento HTML

Es la unidad básica de estructura y contenido de una página web, está compuesta de

  1. Etiqueta de apertura
  2. Pares atributo-valor
  3. Contenido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta que se utiliza para comunicarse con el navegador a tavés de los metadatos permite mejorar el SEO y además en ella se enlazan archivos externos de estilos o scripts

Etiqueta <body>

Define el cuerpo de un documento y contiene todo el contenido visible de una página Web, como texto, imágenes, enlaces y formularios

Contenedores semánticos

Etiqueta Concepto
<header> Representa el contenido introductorio de una página (a menudo contiene un logotipo, un menú de navegación ú otra información de identificación del sitio
<nav> Representa una sección de un pa´gina que contiene enlaces a otra pa´gina o secciones del sitio. Se usa para enlaces de navegacion que ayuda a los usuarios a navegar por nuestro sitio web
<main> Representa el contenido principal de una pagina (no debe incluir ningun contenido que se repita en varias páginas como menús de navegacion, pies de página o barras laterales)
<section> Representa una sección de una página que agrupo contenido relacionado dentro de un sitio web (se puede utilizar para dividir una página en varias seecciones o agrupar contenido dentro una sección más grande)
<article> Representa una unidad autónoma de contenido o historia que puede distribuirse o reutilizarse de forma independiente por ejemplo publicaciones de blogs, articulos de noticias o reseñas de productos
<footer> Representa el contenido de cierre de una página o sección /la parte inferior de una página o sección (a menudo contiene informacion de derechos de autor, detalles de contacto, ubicación ú otra información complementaria)

Elementos principales de HTML

Elemento <a>

Elemento que se utiliza para enlazar o navegar a otras páginas o secciones. Hay 3 tipos de enlaces

  • Externos
  • Internos
  • Enlaces ancla o de anclaje (dentro de nuestra misma página

Algunos valores especiales dentor de su atributo href son

href="https://wa.me/####
Este valor de atributo envia un whatsapp a un número específico
href="mailto:example@gmail.com"
Este valor se utiliza para enviar un correo a una dirección de coreo especifica

Elemento <form>

Es la manera en la que los usuarios se pueden comunicar con una página web, gracias a este elemento se pueden procesar datos de los usuarios y usarlos segun las necesidades u objetivo de nuestro sitio web
